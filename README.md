# Syst-me-cart-sien-
Système de Visualisation Cartésienne Outil performant pour transformer coordonnées et équations en rendus graphiques précis. Inclus : mappage automatique espace-pixels, grille dynamique, gestion du zoom et translation. Idéal pour la géométrie analytique et le tracé de fonctions. Une solution fluide, rigoureuse et facilement extensible.
